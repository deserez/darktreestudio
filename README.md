# Asheron Server Bot Manual

A static command manual for the Asheron Discord bot used by one server.

The site explains:

- what each command means
- when members should use a command or panel
- when staff should use moderation tools
- which admin commands maintain setup, health, config, dashboard, welcome, announcement, and counting systems

## Preview locally

Open `index.html` in a browser, or serve the folder with any static server.

```bash
npx serve .
```

## Publish

This is a plain static site, so it can be hosted on GitHub Pages, Netlify, Vercel, Cloudflare Pages, or any normal web server.

If it is meant only for your Discord server, keep the repository private or share the link only with server members.
